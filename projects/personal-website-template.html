个人网站模板

Personal Website Template System · 可定制的作品集与学术主页模板

面向求职者与研究者两套叙事：前者侧重项目与经历，后者侧重论文与教学；通过可配置版块、导航与视觉（如渐变背景）、技能 SVG 与时间线等模块，在尽量少的代码改动下搭出独立站。配套 README 与入门说明，支持Vercel一键部署。

仓库长期公开维护；具体目录与配置字段以 GitHub 当前代码为准（README 中旧克隆地址 personal-website-temp 已废弃，请使用下方正确仓库名）。

仓库与演示

  • 仓库：https://github.com/songzhiyuan98/personal-website-template
  • 展示站（若与模板同步）：www.zhiyuansong.dev（以仓库 / Vercel 配置为准）

核心特性

  • 双模板版本：求职者版（经历 + 项目）、研究者版（研究 + 教学）
  • 入门引导：交互式说明与文档，降低首次配置成本
  • 关于我：背景与职业目标可配置
  • 技能展示：SVG 图标 + 名称栅格
  • 教育时间线：学历或重要课程
  • 图集版块：照片与高光时刻
  • 高可定制：增删改版块顺序、导航栏与主题色
  • 部署：Vercel 流程与命令行说明

技术栈

快速开始

git clone https://github.com/songzhiyuan98/personal-website-template.git
cd personal-website-template
npm install
  1. 在 app 下打开主要页面（如 page.tsx，以仓库实际路径为准）浏览结构
  2. 选择求职者或研究者模板思路，改版块 id、标题与文案
  3. 填入个人信息、技能与项目 / 论文亮点
  4. 本地预览：npm run dev → http://localhost:3000
  5. 生产部署：安装 Vercel CLI 后执行 vercel，或连接 GitHub 仓库到 Vercel 面板
