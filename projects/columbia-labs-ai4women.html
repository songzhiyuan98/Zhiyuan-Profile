AI4Women 统计门户单页

导师通过邮件布置紧急任务：开发一个以单项统计指标为核心的首页式页面（相对轻量，但需独立成站风格）。截止日期在两周量级内，并希望在截止前能看到可点原型以便迭代文案与配图。

上线策略：按导师要求，该页暂不通过主 LABS 网站「Portal」按钮对公网展示，由团队在内部确认后再调整入口。

品牌：顶栏与底栏需同时呈现 AI4WOMEN 与 LABS 标识；在背景色上尝试与标识协调（含借助模型辅助做配色建议）。

线上站点（公开）： https://labs-laboratory.com/ai4women/ （见 LABS 主站 导航中的 Portals 等入口；页面内容以实验室更新为准。）

视觉与信息架构

与既有 Portal「换皮」不同，本页需要整体风格与配色上突出女性元素与 AI 主题，而不只是替换正文区块。沟通过程中明确了：以配色与导航栏层级传递差异，主内容区（头图、团队介绍等）与导师后续替换的文案/图片解耦，便于对方在终版阶段精修。

先交付多版布局与风格草案，确认方向后再在短周期内完成整页搭建与移动端适配。

交付节奏（摘要）

  • 按周中节点提交 UI 草案，导师反馈「方向正确」后进入实现。
  • 代码托管于 Git 工作分支；优先完成移动端体验再发 PR，避免主分支与导师本地改动冲突（合并前同步主分支）。
  • PR 合并后由导师在版式与文案层精修；后续邮件中提及 TeaChart 等新门户将作为另一条线单独推进。
